# 🖨️ SelfPrintMS: Automated Self-Printing System

[![Live Demo](https://img.shields.io/badge/demo-online-purple)](https://afiqirsyad01.github.io/selfprintms/)
[![Service](https://img.shields.io/badge/Type-SaaS-lightgrey)](#)

SelfPrintMS is a kiosk-style management system that allows users to upload documents and manage print jobs autonomously, reducing the need for manual staff intervention.

## ✨ Key Features
- **📂 Document Upload:** Secure interface for submitting print tasks.
- **⚙️ Print Settings:** Choose Color/BW, Paper Size (A4/A3), and Page Orientation.
- **💳 Credit System:** Integrated balance management for users (PHP-driven).
- **📠 Queue Management:** Real-time visibility of the printer queue.
- **📄 History:** Users can track their previous print jobs and expenses.

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript (File Handling API)
- **Backend:** PHP (Private) - Processes file metadata and cost calculations.
- **Database:** MySQL (Private) - Logs every transaction and print job metadata.

## 🔒 Security & Privacy
The backend is designed to securely handle file temporary storage and automatic deletion after the job is completed to ensure user data privacy.
